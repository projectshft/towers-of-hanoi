Welcome to Towers of Hanoi

by Benjamin Corbett

The goal of the game is to move from array 0 to array 1 or 2.

  • Copy and paste arrMove(board, 0, 0) into the terminal it will be at the top of the console
  • To make it easier press up on your keyboard this willl automatically print the moveArr function
  • After board your first number is index "from" and your second number is index "to"
  • The game is won when all numbers are ordered on peg 1 or peg 2
  • To automatically win and reset type: "winningMoves()" into the terminal
  • Your progress will be tracked through a table system
  • Have fun, the game will be automatically reset after hitting ok on the alert
  • Please refer this Towers of Hanoi for more info about the game
