How to Play

  • The game is in the object called 'game'. To start, type in the console "game.start()"" to see the board.
  • To move a disc from one peg to another use the function "game.move()"". This function takes in two parameters, a "from" and "to" input, in that order.
  • Example: to move a disc from peg one to peg three you would type into the console "game.move(1, 3)".
  • Have fun!!
