Welcome! Use the following instructions to play in the console:

Instructions:

  1. Use board.moveDisc(number, number) to enter the number of a disc's current peg location, followed by a comma and number of the new peg location. Example: board.moveDisc(1, 2) moves top disc from peg 1 to peg 2.
  2. Use stateOfBoard.showBoard() to view board
  3. Use stateOfBoard.resetBoard() to reset game
  4. Use board.help() to see these functions listed in the console
