Towers of Hanoi

If you are unfamiliar with this game, check out this link. This version of Towers of Hanoi utilizes 3 pegs and 5 discs.

Commands

  • moveDisk(startingPeg, endingPeg)
    • Start the game by typing this command in the console. This is the only command you need to play the game. This version only uses three pegs.
  • printBoard()
    • Use this command at the beginning of the game if you wish to view the current state of the board. The board will automatically print to console after each turn.
  • checkWinner()
    • This is already done for you, and will not display a message if you have not yet won.
  • game.count
    • This command will allow you to see your current number of turns.

Good Luck!
