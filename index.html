Towers of Hanoi

William Ashley

Instructions:

  1. Open the JS console in your browser. (alt-command-J)
  2. There is a board consisting of 3 pegs. The first peg contains 5 discs stacked from largest on bottom to smallest on top.
  3. The goal is to move the discs and create the same largest to smallest stack on one of the other pegs.

Rules:

  • Only one disc can be moved at a time.
  • Only the top disc on the peg can be moved.
  • A larger disc cannot be placed on top of a smaller disc.

Controls:

  • Use the 'moveDisc(fromPeg, toPeg);' function to move the discs around the board.
    • Ex: moveDisc(1, 2); //Will move the top disc from Peg 1 to Peg 2
  • To play with more pegs and/or discs, create a new board by using 'boardObj.createBoard(numPegs, numDiscs);'
    • Ex: boardObj.createBoard(4, 7); //Creates a board with 4 pegs and 7 discs
  • If you get stuck or would like to start over, type 'boardObj.resetBoard();'
    • This will start the game over and reset the board back to the default 3 pegs with 5 discs board.
